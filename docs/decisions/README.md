# Architecture decisions

- **Status:** Active route
- **Kind:** Rationale archive
- **Authority:** Canonical for accepted decision history only
- **Owns:** Why durable choices were made and which alternatives were rejected
- **Does not own:** Current behavior; direct owners in `docs/INDEX.md` remain authoritative

Decision records use `NNNN-short-name.md`, state a status and date, and link current owners. Change
behavior by updating its owner first, then supersede the ADR when its rationale no longer applies.
